# コマンド

## チェックアウト

```
$ git checkout <branch>
```

##  変更したファイルを全てステージング

```
$ git add -A
```

## メッセージをつけてコミット

```
$ git commit -m "ここにコミットメッセージ"
```

##  プッシュ
```
$ git push
```
